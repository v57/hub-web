Hub
Publish, connect, stream. All in one Hub .
Hub is a local-first gateway for services. Publish APIs once, expose them through one endpoint. Use one protocol for request and stream. Lite is for isolated local use. Pro adds keys, groups, permissions, proxy, and merge. Less glue code, fewer “which port is this?” moments.

How Hub works
Worker SDK
Hub Service lets each worker publish clean, secure APIs that your hub can call.
Production control
Hub Pro adds team controls, permissions, and safer routing for bigger installations.
Services SDK
Hub Lite helps register workers and sends jobs to the right service automatically.
Transport
Channel handles message flow across request, stream, and event patterns, with automatic reconnect support.
Operations
Hub Launcher packages app deployment and running process controls into one simple control area.
